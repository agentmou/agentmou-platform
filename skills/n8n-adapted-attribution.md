## Adaptation Notes

The repo-local versions are adapted for AgentMou rather than mirrored
verbatim.

Adaptations include:

- AgentMou-specific workflow-promotion rules
- repo-truth vs runtime-truth guardrails
- references to local docs and templates
- guidance for Codex and Cursor in this repo
- reduced scope so the bundle stays focused on natural-language workflow
  authoring through MCP
