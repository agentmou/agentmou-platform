# Traefik

Traefik is configured entirely through labels and CLI arguments in
`infra/compose/docker-compose.prod.yml`. There is no separate `traefik.yml`
file in this repository.

## Purpose

- Terminate TLS for the VPS-hosted services
- Apply shared middlewares such as auth, security headers, and rate limiting
- Persist ACME certificate state across container restarts

## Directory Structure

```text
traefik/
└─ letsencrypt/
   └─ acme.json   # Auto-generated by Traefik; never commit
```

## Notes

- The `letsencrypt/` directory is bind-mounted into the Traefik container.
- `acme.json` is generated at runtime and must stay out of git.
- Route intent and middleware details live in
  [`docs/runbooks/vps-operations.md`](../../docs/runbooks/vps-operations.md).
