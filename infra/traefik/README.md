# Traefik

Traefik is configured entirely via CLI arguments in
`infra/compose/docker-compose.prod.yml`. There is no `traefik.yml` config
file.

## Directory structure on the VPS

```
traefik/
└── letsencrypt/
    └── acme.json   # Auto-generated by Traefik — do NOT commit
```

The `letsencrypt/` directory is bind-mounted into the Traefik container so
TLS certificates persist across restarts. The `acme.json` file is
auto-generated and must never be committed to git.
